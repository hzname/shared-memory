# Hermes

CLI AI-агент. Работает в WSL (Ubuntu). Основной ассистент.

## Возможности

- MCP-клиент (встроенный)
- Skills: ~/.hermes/skills/
- Memory: persistent memory tool
- Shell, browser, file tools
- Cron, TTS, image gen

## Роль

Основной ассистент для робототехники и продуктов.
Подключается к shared-memory через MCP.
