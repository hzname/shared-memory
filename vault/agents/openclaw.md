# OpenClaw

Self-hosted AI-агент gateway. Node.js.

## Возможности

- MCP-клиент и MCP-сервер
- Skills, Plugins, Hooks
- Memory: MEMORY.md + semantic search (BM25+vec)
- SubAgents, ACP agents
- Multi-channel: Telegram, Discord, Slack и др.

## Роль

Мульти-канальный ассистент с мощной системой памяти.
