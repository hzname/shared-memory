# ЭСРМ

Электромеханический модуль специального назначения.

## Ключевые решения

- АКБ = пассивное охлаждение (без активного обдува)
- ВПУ = чистый вакуум
- СРМ = один ротор, две спирали
- ЩЗ = «Щетка зимняя»
- УМ-У = контроллер (не «навесное оборудование»)
- Без фразы «Навешивается на УМ-У»
